Lesson 1.1: Introduction to Python Syntax

What is Syntax?

In programming, syntax refers to the set of rules that define how you write code in a given language. Python syntax is designed to be clean and readable, making it a great first language for beginners.

Example: Print a Message

The simplest Python statement is the print() function. Try it below:


  


  
    from js import document
    
    def run_python_code(event):
        code = document.getElementById("code").value
        output = document.getElementById("output")
        output.innerText = ""
        try:
            exec(code, {"print": lambda *args: output.innerText += " ".join(map(str, args)) + "\\n"})
        except Exception as e:
            output.innerText = f"Error: {e}"
    
    document.getElementById("run-btn").addEventListener("click", run_python_code)
