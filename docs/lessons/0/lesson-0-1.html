Lesson 0.1: Setting Up Your Development Environment

Before we can start learning how to write Python code, we need the right tools. In this lesson, we’ll set up your development environment so you’re ready to run, test, and build Python programs confidently throughout the rest of the book.

What is a Development Environment?

A development environment is the collection of tools you use to write and run your code. This includes things like a code editor, a terminal, and the Python interpreter itself.

The Tools You’ll Be Using

1. Python (Interpreter)

Python is the language we’re learning. To check if it’s already installed, open your terminal and type:

python --version

If you don’t see a version number, install Python using one of the methods below.

Installing Python

Option 1: Microsoft Store (Windows)

When opening a .py file in VS Code, you might see this prompt if Python is missing:

Click the prompt to open the Microsoft Store page for Python:

Click Get to install Python quickly and easily.

Option 2: Install from python.org

Visit python.org/downloads and download the installer for your operating system:

Run the installer. Be sure to check Add Python to PATH before clicking Install Now:

2. Visual Studio Code (Editor)

  • Download from: code.visualstudio.com
  • Used for writing, editing, and running your Python scripts.

3. Python Extension for VS Code

  • Press Ctrl + Shift + X in VS Code
  • Search for and install the "Python" extension by Microsoft

4. Terminal or Command Prompt

Open your terminal in VS Code by pressing Ctrl + `.

Testing Your Setup

  1. Create a new file called hello.py
  2. Type:
print("Hello from CyberNet!")
  1. Run the file using the terminal:
python hello.py

You should see:

Hello from CyberNet!

Why These Tools?

  • Free and open source
  • Cross-platform (Windows, macOS, Linux)
  • Used professionally in many industries
  • Flexible and creative — perfect for building interactive learning experiences

Summary

  • Install Python, VS Code, and the Python extension
