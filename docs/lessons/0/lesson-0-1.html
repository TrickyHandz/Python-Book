Lesson 0.1: Setting Up Your Development Environment

Before we can start learning how to write Python code, we need the right tools. In this lesson, we’ll set up your development environment so you’re ready to run, test, and build Python programs confidently throughout the rest of the book.

What is a Development Environment?

A development environment is the collection of tools you use to write and run your code. This includes things like a code editor, a terminal, and the Python interpreter itself.

Think of it like your digital workbench—it should be organized, reliable, and ready for creative experimentation.

The Tools You’ll Be Using

1. Python (Interpreter)

Python is the language we’re learning. It’s an interpreted language, which means we’ll use the Python runtime to run our .py files.

You should already have Python installed on your machine. To verify, open your terminal and type:

python --version

You should see something like:

Python 3.11.6

Installing Python

If you don’t see a version number, Python may not be installed yet. Here’s how to install it:

Option 1: Microsoft Store (Windows)

  1. If you open a .py file in VS Code and Python is missing, it will prompt you to install from the Microsoft Store.
  2. Click the prompt, and install Python 3.x from the store.
  3. After installation, restart VS Code.
  4. Verify installation:
python --version

Option 2: python.org (Windows, macOS, Linux)

  1. Visit python.org/downloads
  2. Download the latest version for your OS.
  3. Run the installer.

Important (Windows): Be sure to check "Add Python to PATH" before you click Install.

After installation:

python --version

or if that fails:

python3 --version

2. Visual Studio Code (Editor)

VS Code is our code editor of choice. It’s lightweight, powerful, and highly customizable.

  • Download it from: code.visualstudio.com
  • We’ll use it to write code, open terminals, and run scripts.

3. Python Extension for VS Code

  • Open the Extensions view in VS Code (Ctrl + Shift + X)
  • Search for “Python” and install the extension by Microsoft

4. Terminal or Command Prompt

The terminal is how you’ll run Python from the command line. In VS Code, press Ctrl + ` to open the terminal.

Testing Your Setup

  1. Open VS Code
  2. Create a new file: hello.py
  3. Type:
print("Hello from CyberNet!")
  1. Save the file
  2. Open the terminal and run:
python hello.py

You should see:

Hello from CyberNet!

Why These Tools?

  • Free and open source
  • Cross-platform (Windows, macOS, Linux)
  • Used professionally in many industries
  • Flexible and creative — perfect for building interactive learning experiences

Summary

  • Install Python, VS Code, and the Python extension
  • Use VS Code to write and run Python code
  • Verify your setup by running a test program
